##  What I'm Building

-  **[aura-agent](https://github.com/BraianTroncoso/aura-agent)** `open source · MIT` — Your personal AI agent à la *Her*, living in Telegram: it reads your email, calendar and assigned PRs and writes your daily standup, all on a **local** model so your data never leaves your machine. *Not a chatbot. A second mind.* *(FastAPI + SQLite + Ollama + Telegram)*

-  **[fracarg](https://github.com/BraianTroncoso/fracarg)** `private` — An Argentine remix of the 1997 DOS classic *Fracas*: a wizard Bomberman with exploding cauldrons, story bosses and survival waves over a deterministic, server-ready game core. *Not a clone. A nostalgia engine.* *(Phaser 3 + TypeScript + Vite + Supabase)*

-  **[DevLog](https://github.com/BraianTroncoso/devlog)** — A living, bilingual logbook of applied AI: a 12-module *school* (Fundamentals → Production) plus a *Radar* feed that keeps every module current. *Not a course that ages.* *(Astro + Starlight + Mermaid)*

-  **[archetype-lab](https://github.com/BraianTroncoso/archetype-lab)** — A behavioral-physics lab where you manipulate 8 human archetypes under simulation-theory rules, with deterministic RNG forking and Gödel-style self-blind agents. *Not a sandbox. A hypothesis test.* *(TypeScript + Vue 3 + bitECS + PixiJS 8)*

-  **[Frontdeck](https://github.com/BraianTroncoso/Frontdeck)** `private` — A design-system platform that generates premium sites (each with its own identity and architecture) from one codebase: 474+ Vue components, 22 themes, 200+ variants. *Not a template. A design engine.* *(Laravel 12 + Vue 3 + Filament + Three.js + Tailwind 4)*

-  **[Cortexa](https://github.com/BraianTroncoso/cortexa)** `private` — Turns your team's knowledge into AI agents with judgment and tone, cloning real collaborators via `SOUL.md` heuristics; ships a 6-agent fintech demo with compliance audits. *Not a chatbot. A digital twin of your company.* *(Paperclip + Ollama + Marp)*

-  **[LevelUp](https://github.com/BraianTroncoso/LevelUp)** `private` — A stoic habit-tracker PWA: 23 daily habits, an honest self-evaluation flow, and live chat with Marcus Aurelius, Seneca and Epictetus (Llama 3.3). Trilingual, offline-capable. *(Next.js 16 + Groq + PWA)*

-  **[Voicepen](https://github.com/BraianTroncoso/Voicepen)** `private` — Lend your voice to an agent and it writes your LinkedIn/Instagram posts in your exact tone, grounded on fresh news so it never invents facts. *Not a generic AI writer. A pen that knows your voice.* *(Next.js 16 + Drizzle/Turso + Groq)*

-  **[takeclass](https://github.com/BraianTroncoso/takeclass)** — Turns your day's git diff into an English-practice session: vocabulary, a reading script and drills from the code you actually shipped today. *(Claude Code skill + slash command · MIT)*

-  **[myjarbis](https://github.com/BraianTroncoso/myjarbis)** — Gives Claude Code persistent memory and structured workflows via MCP: one global server, per-project memory, code search and a CLI. Install once, use everywhere. *(Node.js + MCP)*

-  **[GesturePilot](https://github.com/BraianTroncoso/GesturePilot)** — Control Claude Code with hand gestures and voice from your webcam: MediaPipe hand-tracking + local Whisper + Piper TTS, no keyboard needed. *(Python + MediaPipe + Faster-Whisper)*

-  **[IwannaWork](https://github.com/BraianTroncoso/IwannaWork)** `private` — Job-hunting autopilot for LinkedIn Easy Apply: applies through multi-step forms via shadow-DOM traversal, with an adaptive throttle that mimics a tired human, not a ghost. *(Python + FastAPI + Vue 3 + Selenium)*

-  **[GymSphere](https://github.com/BraianTroncoso/GymSphere)** `private` — Gym-management platform: members, plans, attendance and billing in one place.

-  **[SecondBraian](https://github.com/BraianTroncoso/SecondBraian)** `private` — A personal knowledge vault in Obsidian (PARA + Maps of Content, 10 templates), where chaos becomes clarity. *(Obsidian)*

-  **[GAME-JAM-FAD-CUYO-2022](https://github.com/BraianTroncoso/GAME-JAM-FAD-CUYO-2022)** — A complete game built in under 12 hours at a game jam. Ship or die.

-  **[trainme](https://github.com/BraianTroncoso/trainme)** `open source · MIT` — A tiny CLI that preps you for an interview from your *persona* + a job posting: a 1-hour roadmap, likely Q&A and stack tips, no retyping. *Not a prompt you retype. A CLI that preps you in minutes.* *(Python + Rich + pypdf)*
